# Pokedex

Este proyecto tiene por objetivo ser el primer acercamiento al uso de JavaScript en una aplicación web.

En un primer instante, se busca realizar la consulta de la PokeAPI (https://pokeapi.co/) para consultar los datos de los pokemón que contiene, donde se muestre la siguiente información:

+ Nombre
+ Imágen
+ Tipo de pokemón
+ Estadísticas
+ Movimientos

La información se despliega en la página por medio de una búsqueda que considera el nombre del pokemón como variable.

La maquetación de la página y el diseño con CSS está basado en el diseño de VMRoldao, además de tomar como base el código script.js y html de Manuel Basanta disponible en github.
